# v3.2 Bunfig Validation - Quick Reference

## Core Commands

```bash
# Full validation (14ms)
bun validate:bunfig

# Full rules validation (includes bunfig)
bun rules:validate

# Grep config errors (red highlight)
bun grep:config-errors

# Build error index
bun index:config-errors
```

## Git Integration

### Install Pre-Commit Hook

```bash
# Copy hook to .git/hooks
cp scripts/git-hooks/pre-commit .git/hooks/pre-commit
chmod +x .git/hooks/pre-commit
```

### Auto-Validation on Commit

```bash
# Hook automatically runs bun validate:bunfig
git commit -m "fix: update editor"
```

## Error Index Queries

```bash
# Build error index first
bun index:config-errors

# Search for specific keys in files with errors
rg "editor" $(cat .bunfig-errors.index)
rg "seed" $(cat .bunfig-errors.index)
rg "timeout" $(cat .bunfig-errors.index)
```

## PR Preparation

```bash
# Auto-generate branch and validate
bun rules:pr BUNFIG-VALIDATE
```
